Html Introduction

HTML stands for Hyper Text Markup Language . It is a standard markup language for creating web sites .


HTML describes the structure of the web page. It consistsof a series of elements.


HTML elements tell the browser how to display the content. It label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

Web Browsers

Web browser like Chrome, Edge, Firefox, Safar , etc is use to read HTML documents and display them correctly.

A browser does not display the HTML tags, but uses them to determine how to display the document

HTML Page Structure

<html>

<head>

<Title>Page Title</Title>

<head>

<body>

<h1>Heading</h1>

<p>Paragraph</p>

<p>Paragraph</p>

</body>

</html>
